# 📚 Sistema de Biblioteca (Java)

Um sistema simples de biblioteca desenvolvido em **Java**, com foco em **Programação Orientada a Objetos (POO)**.

---

## 🚀 Funcionalidades

### 📖 Livros
- Cadastrar livros
- Listar livros
- Atualizar informações
- Remover livros

### 👤 Usuários
- Cadastrar usuários
- Listar usuários
- Atualizar dados
- Remover usuários

### 🔄 Empréstimos
- Realizar empréstimo de livros
- Listar empréstimos ativos
- Devolver livros

---

## 🧠 Conceitos de POO aplicados

- ✔️ Encapsulamento  
- ✔️ Associação entre classes  
- ✔️ Separação de responsabilidades (Service / Model)  
- ✔️ Manipulação de objetos  
- ✔️ Estrutura modular  

---

src/
